# Random Password Generator

**Автор:** Бурцев Константин

**Описание:**
Простое приложение с графическим интерфейсом для генерации случайных паролей. Позволяет настраивать длину и состав символов (цифры, буквы, спецсимволы). История последних 10 сгенерированных паролей сохраняется в файл и отображается в таблице.

**Как использовать:**
1. Запустите `password_generator.py`.
2. Установите желаемую длину пароля с помощью ползунка.
3. Выберите типы символов с помощью чекбоксов.
4. Нажмите кнопку «Сгенерировать».
5. Сгенерированный пароль появится в поле ввода и будет добавлен в историю.
6. История сохраняется автоматически при закрытии программы.

**Примеры использования:**
- Сгенерировать сложный пароль из 16 символов: установите длину на 16 и отметьте все чекбоксы.
- Сгенерировать простой PIN-код из 4 цифр: установите длину на 4 и отметьте только «Цифры».
- Проверка: программа не позволит сгенерировать пароль без выбранных типов символов.
